## Tech Stack

**Client:** React, Redux, MaterialUI, React DnD, 

**Server:** NestJS, MongoDB

## Features

- Acces and refresh token authentication

- Create new users

- RBAC authorization

- Drag and drop task management

- Light/dark mode toggle
